PART 2 · Chapter 04

확률변수와 기댓값 — 무작위를 숫자로

"앞/뒤", "비/맑음" 같은 결과에 숫자를 붙이면(확률변수) 평균·합·비교를 계산할 수 있어요. 그 중심값이 기댓값(장기 평균), 퍼진 정도가 분산이에요. 이 둘이 분포를 요약하는 가장 기본 숫자예요.

Step 00 — Random Variable

결과에 숫자를 붙인다

확률변수(random variable)는 "무작위 결과 → 숫자"로 보내는 규칙이에요. 숫자가 되면 더하고 평균낼 수 있죠.

예: 주사위 눈

결과 {⚀…⚅} → 값 {1,2,3,4,5,6}. 각 1/6 확률.

예: 동전 앞면 수

동전 2개 → 앞면 개수 {0,1,2}. 확률 {1/4, 2/4, 1/4}.

각 값에 확률이 붙은 표가 확률분포예요(모든 확률 합=1). 분포를 한두 숫자로 요약하는 게 기댓값(중심)과 분산(퍼짐)이고요.

→ 이 장은 값이 띄엄띄엄한 이산 확률변수로 시작해요. 연속(키·몸무게)은 6장 정규분포에서.

Step 01 — Expected Value

기댓값 = 값 × 확률을 모두 더한 것

주사위 눈의 기댓값 E[X] = 1·⅙ + 2·⅙ + … + 6·⅙ = 3.5. "한 번에 3.5는 안 나오지만, 많이 굴린 평균이 3.5로 간다"는 뜻이에요(큰 수의 법칙). 직접 굴려 확인해요.

세로축 = 누적 평균 · 점선 = 기댓값 3.5

→ 표본 평균이 기댓값 3.5로 수렴하죠. 기댓값은 "이론적 장기 평균", 표본 평균은 "실제로 재본 평균" — 많이 모으면 둘이 만나요.

Step 02 — Variance & Spread

같은 평균, 다른 퍼짐

기댓값이 같아도 분포 모양은 천차만별이에요. 셋 다 평균 3.5인데 퍼진 정도(분산)가 달라요. 분산 = "평균에서 떨어진 거리를 제곱해서 평균낸 것"이에요.

왜 제곱할까요? 그냥 거리를 더하면 평균보다 큰 쪽(+)과 작은 쪽(−)이 상쇄돼 늘 0이 돼요. 제곱하면 부호가 사라지고, 큰 차이가 더 크게 반영돼요. 대신 단위가 "점²"처럼 제곱이 되니, √로 되돌린 게 표준편차(원래 단위)예요.

막대 = 각 값의 확률 · │ = 평균(3.5)

→ "3 또는 4"는 평균 근처에 몰려 분산이 작고(0.25), "1 또는 6"은 양극단이라 분산이 커요(6.25). 평균만으론 분포를 다 알 수 없어요 — 퍼짐(분산)도 봐야 하죠. 리스크·변동성이 다 분산 이야기예요.

Step 03 — Expected Value in Action

복권은 왜 사면 손해일까

기댓값은 "이 게임을 무한히 반복하면 한 판 평균 얼마"예요. 기대 순이익 = (당첨확률 × 당첨금) − 표값. 여기선 표값을 1만원으로 고정하고, 당첨 확률과 당첨금만 조절해봐요(당첨금 슬라이더는 만원 단위, 화면엔 억으로 표시).
