# 확률과 통계, 눈으로 보기 (Probability & Statistics, Visualized)

확률부터 가설검정까지 — 불확실함을 다루는 언어를 **동전·주사위·분포·표본**으로 직접 던지고 뽑으며 익히는 인터랙티브 학습 자료입니다. 모든 표본추출·분포·추정·검정이 브라우저에서 실제 로직으로 동작합니다.

> 직접 만든 **비영리 학습용 자료**입니다. 특정 교재·강의와 무관하며, 설명·예시·코드는 모두 독자적으로 제작했습니다.

## 설계 원칙

- **실제 계산**: 표본추출·분포·추정·검정이 모두 실제 코드로 동작 (하드코딩 결과 없음)
- **직관 먼저, 함정 직시**: 큰 수의 법칙·CLT를 눈으로, 그리고 p-value·base rate 같은 흔한 오개념을 정직하게 다룸
- **단계별 분해**: 각 챕터 4스텝, 한 줄 직관 → 인터랙션 → 결론
